# Proyecto - Segunda Entrega

Este proyecto es una práctica en Node.js con Express.

## Requisitos

Asegúrate de tener instalado:
- [Node.js](https://nodejs.org/) (versión recomendada: 18.x o superior)
- [npm](https://www.npmjs.com/) (incluido con Node.js)

## Instalación

1. Clona este repositorio:

   git clone <https://github.com/augusto66/Backend_1.gitURL_DEL_REPOSITORIO>
   cd primer_entrega

2. Instala las dependencias:
   npm install
   

## Dependencias utilizadas

Las principales dependencias de este proyecto son:
- **Express**: Framework para crear el servidor web.
- **UUID**: Generador de identificadores únicos universales (UUID).

Instalación manual de dependencias si es necesario:
npm install express uuid
## Ejecución del proyecto

Para iniciar el servidor en modo desarrollo:
npm run dev

El servidor estará disponible en `http://localhost:8080`.

## Archivos importantes
- `src/server.js` → Archivo principal del servidor.
- `src/routes/` → Contiene las rutas del proyecto.
- `src/controllers/` → Lógica de negocio.
